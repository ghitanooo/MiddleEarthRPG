## How to Play
| Key | Action |
|-----|--------|
| **ENTER** | Select / confirm |
| **A** | Attack in battle |
| **I** | Open inventory |
| **B** | Begin your next adventure |
| **R** | Restart after defeat |
| **ESC** | Quit the game |



---

## Quick Play (Windows)

1. Go to the [**Releases**](https://github.com/ghitanooo/MiddleEarthRPG/releases) section.  
2. Download **MiddleEarthRPG.exe** from the latest release.  
3. Run the file and enjoy your journey!
